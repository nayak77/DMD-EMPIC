# Accelerating Particle-in-Cell Kinetic Plasma Simulations via Reduced-Order Modeling of Space-Charge Dynamics using Dynamic Mode Decomposition

## Overview
This repository contains the MATLAB code developed by Indranil Nayak for the paper titled "Accelerating Particle-in-Cell Kinetic Plasma Simulations via Reduced-Order Modeling of Space-Charge Dynamics using Dynamic Mode Decomposition." The paper has been accepted for publication in APS Physical Review E.

**Authors:** Indranil Nayak, Fernando L. Teixeira, Dong-Yeop Na, Mrinal Kumar, and Yuri A. Omelchenko.

The preprint is available at: [https://arxiv.org/abs/2303.16286](https://arxiv.org/abs/2303.16286)
